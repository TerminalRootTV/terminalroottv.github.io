O que você vai aprender ?

sse curso foi projetado para quem não sabe nada e também para aqueles que desejam reaprender da forma certa ou evoluir. Conhecimentos sólidos de banco de dados SQL e Programação Orientada à Objetos que servirão para facilitar o aprendizado em qualquer outra linguagem que a pessoa desejar aprender. PHP é a linguagem com a base mais fácil para entender Orientação à Objetos e integração com bases de dados. Além de PHP & MySQL, esse curso inclui também as tecnologias:

  • HTML5
  • CSS
  • Javascript
  • jQuery
  • Bootstrap
  • Jekyll
  • Github, Gitlab, Netlify
  • Programação Orientada à Objetos

Além de Laravel o framework mais prestigiado entre os programadores PHP. PHP-GTK para criar sistemas locais integrados. Diversas dicas de Apache, configurações avançadas. E tudo isso incluso dicas de permissões e utilização de comandos no Linux. Veja a grade completa.
